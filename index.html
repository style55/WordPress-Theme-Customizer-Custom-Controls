WordPress тема Настройщик пользовательских элементов управления Тема настройщик WordPress позволяет изменить стиль и функциональность вашей темы WordPress и увидеть изменения, внесенные в режиме реального времени без сохранения тему WordPress. Вот коллекция пользовательских элементов управления, которые можно использовать на вашем тематическом настройщика. Пока эта коллекция состоит из: - Категория Выпадающее - Создает выпадающий всех категорий по вашей теме WordPress. - Выбор даты - Добавляет контроль сборщика даты в тематический настройщика. - Макет Выбор - Добавляет 3 изображения макетов к странице выбрать новый стиль. - Выпадающее меню - Создает выпадающий всех меню на вашем сайте WordPress. - Сообщение Выпадающее - Создает выпадающий всех сообщений на вашем WordPress сайте. - Ключевые слова Выпадающее - Создает выпадающий всех тегов на вашем сайте WordPress. - Текстовый редактор - Создает текстовое с текстовой TinyMCE. - Textarea - Создает поле ввода текстовой области - Систематика Выпадающее - Создает выпадающий таксономии (Использование: см Суть 4538951) - Список пользователей Выпадающее - Создает выпадающий пользователей для роль (Использование: см Суть 4564337)
